Chapter 6. HTML5 Plugin

The HTML5 transformation type produces HTML5-based output that integrates newer web technologies, such as HTML5, CSS3, responsive layouts.

The HTML5 transform provides dynamic and modern navigation and presentation. Supporting components, such as the CSS style sheets and JavaScript us ed to render the HTML in browsers, that are designed to be easily adapted or replaced as required to provide specific navigation and presentation styles.

The HTML5 plugin contains a "theme mechanism". The theme mechanism provides a way to register, concatenate and compress css and javascript into a single theme.

Note: The HTML5 plugin does not use the HTML frames mechanism.
