The default theme

  • Replacing the index.html file
    The HTML5 transformation type does not produce an index.html file by default.
  • Multiple columns layout
    The html5plugin can allows you to use the latest CSS multi-columns layout for the browsers that support it.
  • CSS grid
    The html5 plugin use the 960 Grid System, which allows technical writters to use the attribute outputclass to positions their elements on the grid.
  • Image support
    The plugin offers the different options for images.
  • Notes
    Notes are output with the aside element except when @importance is set to low or obsolete, which will output a details elements. Details elements should be collapsed by default on new browsers.
  • UI
    The plugin allows to interact with jQuery UI to output UI on the content.
